# Zero engine

## Установка заисимостей

```
sudo apt install libsfml-dev
```

## Компиляция
```
make compile
```

## Запуск движка
```
make run
```

## Запуск редактора карты
```
make editor-run
```

## Управление
WASD - движение
LeftArrow/RightArrow - поворот камеры
R - смена режима отображения


## Редактор карты
lclick - создать тайл
rclick - удалить тайл
Enter - сохранить карту
